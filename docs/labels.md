# Labels

Labels are how we turn issues into a contributor operating system.

## Area Labels

- `area: core-engine` - pure prompt, screenplay, event planning, and canon primitives.
- `area: service-worker` - Redis/BullMQ worker and service contracts.
- `area: nft-pipeline` - mint art, state-loop APNGs, reroll clips, cycle recaps, and the APNG assembler.
- `area: trailer-pipeline` - trailer script passes, generation, assembly, and canonization.
- `area: prompts` - dialogue, keyframe, motion, negative prompt, and director grammar work.
- `area: video` - image-to-video, lip-sync, pacing, crop safety, and artifact issues.
- `area: world-pack` - country, faction, character, location, voice, and story memory packs.
- `area: providers` - model, storage, delivery, and media provider adapters.
- `area: evals` - proof manifests, scorecards, linting, fixtures, and regression checks.
- `area: docs` - README, docs, examples, and contributor guidance.
- `area: automation` - GitHub Actions, issue templates, labeler, and repo automation.

## Proof Labels

- `proof: needed` - needs media proof, fixture proof, or reproduction proof.
- `proof: supplied` - contributor supplied proof, not yet reviewed.
- `proof: sufficient` - maintainer agrees proof is enough for the touched surface.
- `proof: clip` - includes generated video proof.
- `proof: frame` - includes generated image/keyframe proof.

## Quality Labels

- `quality: character-consistency`
- `quality: dialogue`
- `quality: motion`
- `quality: lip-sync`
- `quality: story-continuity`
- `quality: brand-fit`
- `quality: artifacts`

## Contributor Labels

- `good first issue` - safe for newcomers.
- `help wanted` - maintainers want outside contributions.
- `bounty` - contribution may be eligible for public reward/credit.
- `ai-assisted` - PR was built with AI assistance and should include enough context for review.

## Priority Labels

- `P0` - blocks production/content release or risks secrets/data.
- `P1` - high-priority broken workflow or serious quality regression.
- `P2` - normal backlog issue.
- `P3` - polish, docs, examples, nice-to-have.
